Statistiques

1. Statistiques descriptives

Indicateurs de position

Moyenne (μ) = ∑x/n
Médiane = valeur qui sépare la série en deux parties égales
Mode = valeur la plus fréquente
Quartiles = Q1 (25%), Q2 (50% = médiane), Q3 (75%)

Indicateurs de dispersion

Variance (σ²) = ∑(x-μ)²/n
Écart-type (σ) = √(variance)
Étendue = valeur max - valeur min
Écart interquartile = Q3 - Q1

2. Distributions statistiques

Distribution normale

Caractéristiques :

  • Forme en cloche symétrique
  • Moyenne = médiane = mode
  • 68% des données entre μ±σ
  • 95% des données entre μ±2σ
  • 99.7% des données entre μ±3σ

3. Probabilités

Concepts de base

  • P(A) = probabilité de l'événement A
  • 0 ≤ P(A) ≤ 1
  • P(A∪B) = P(A) + P(B) - P(A∩B)
  • P(A|B) = P(A∩B)/P(B)

Variables aléatoires

Espérance (E(X)) = moyenne théorique
Variance (V(X)) = E((X-E(X))²)
Écart-type = √V(X)

4. Échantillonnage

Théorème central limite

La distribution des moyennes d'échantillons tend vers une loi normale quand n augmente.

Erreur-type = σ/√n
Intervalle de confiance à 95% = x̄ ± 1.96×(σ/√n)

5. Tests statistiques

Test de Student (t-test)

Compare la moyenne d'un échantillon à une valeur théorique ou deux moyennes entre elles.

  • H0 : hypothèse nulle
  • H1 : hypothèse alternative
  • p-value : probabilité d'obtenir un résultat au moins aussi extrême

Test du Chi-deux

Teste l'indépendance entre deux variables qualitatives.

χ² = ∑((O-E)²/E)
O = effectifs observés
E = effectifs théoriques

6. Corrélation et régression

Coefficient de corrélation (r)

r = Cov(X,Y)/(σx×σy)
-1 ≤ r ≤ 1
r = 1 : corrélation positive parfaite
r = -1 : corrélation négative parfaite
r = 0 : absence de corrélation linéaire

Régression linéaire

y = ax + b
a = Cov(X,Y)/V(X)
b = ȳ - ax̄
R² = coefficient de détermination
